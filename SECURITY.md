# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| v1.0.0 | Yes |

## Reporting Security Issues

Please do not open public issues for security vulnerabilities.

Report security concerns privately to:

Freshtiq Innovations  
freshtiqinnovations@gmail.com

## Security Practices

- Do not commit `.env` files.
- Do not commit API keys, bot tokens, passwords, private keys, or session files.
- Keep runtime logs and backups outside Git.
- Rotate any exposed token immediately.
- Use environment variables for secrets.
- Review changes before deployment.
